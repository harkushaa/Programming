# Программирование (2 семестр)

## Описание дисциплины
Дисциплина "Программирование" (2 семестр) направлена на изучение объектно-ориентированного программирования на языке C#. В рамках курса рассматриваются основные принципы ООП, работа с классами, наследование, полиморфизм, интерфейсы, паттерны проектирования и другие ключевые концепции.

## Содержание лабораторных работ

### Лабораторная работа №1
**Тема:** Ввод/вывод данных и арифметические операции

Простая программа для деления двух чисел с выводом результата.

### Лабораторная работа №2
**Тема:** Условные операторы и циклы

Программа с двумя задачами: сравнение цифр числа и проверка попадания координат в область.

### Лабораторная работа №3
**Тема:** Методы и ветвление

Вычисление функций, разветвляющиеся алгоритмы, работа с датами (день недели, разница между датами).

### Лабораторная работа №4
**Тема:** Singleton паттерн. Классы и объекты

Реализация сервиса ЖКХ с использованием паттерна Singleton для управления тарифами и расчета задолженности.

### Лабораторная работа №5
**Тема:** Коллекции. Работа со списками

Система управления жильцами и услугами: добавление, поиск, расчет стоимости с учетом тарифов.

### Лабораторная работа №6
**Тема:** Наследование и полиморфизм

Иерархия классов алкогольных напитков с абстрактным базовым классом и переопределением методов.

### Лабораторная работа №7
**Тема:** Перегрузка операторов. Интерфейсы

Класс интервала с перегруженными арифметическими операторами, операторами сравнения и реализацией интерфейса IComparable.

### Лабораторная работа №8
**Тема:** Интерфейсы. Принципы SOLID

Система расчета коммунальных платежей с использованием интерфейсов для определения скидок.

### Лабораторная работа №9
**Тема:** Паттерны проектирования (Builder, Factory)

Агентство недвижимости с применением паттернов Builder и Factory для создания различных типов риелторов.

## Технологии
- **Язык:** C# (.NET 8.0)
- **Среда разработки:** Visual Studio
